# MarkDown 基本语法

1、标题

  最小为六级字体（#号）

2、字体

  **加粗**
  *倾斜*`
  ***斜体加粗***
  ~~删除线~~

3、引用（备注功能）

>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容

4、分割线(显得好看一些)

---
***
三个或者三个以上的*号/-号 效果等同

5、图片

![alt](图片地址 ''图片title'')
alt就是显示在图片下面的文字，相当于对图片内容的解释。
title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加

6、链接

[链接名](链接地址,'标题可不写')

7、无序列表

下列三个符合效果等同 一定注意有空格

- 内容1
+ 内容2
* 内容3

8、有序列表

数字加点

1. 内容1
2. 内容2

9、列表嵌套

一级列表和二级间隔三个空格

1. 内容1
   * 内容1.1

10、表格

冒号：加在什么位置 文字居左/右
两边都加则居中

表头|表头|表头
---|:--:|---:
内容|内容|内容

11、代码块

1. 单行代码用一个`包裹起来
2. 块代码用三个`包裹起来
