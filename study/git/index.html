# Git 基本使用!

介绍几个常用的基本Git命令 文档中反单引号只是高亮显示 不是命令的一部分 完善中ing

1、创建并切换到分支

git checkout -b `local-branch` origin/`remote-branch` （远程的分支名）

2、添加代码到暂存区

git add `.`    //添加当前项目的所有文件到暂存区
        `dir`  //添加某个文件到暂存区

3、暂存区代码提交到仓库区

git commit -m 'msg'

4、更新(拉取)代码

git pull ===  git fetch + git checkout
git pull origin `remote-branch`

5、提交(push)代码

git push -u origin `remote-branch` //适用于第一次提交 未指定远程分支的情况
git push  == git push origin `remote-branch` //已指定远程的分支

6、撤销

git checkout `file` //撤销文件到工作区  文件指暂存区及之前的文件
git checkout `commit` `file`  //撤销文件到暂存区和工作区
git checkout `.` //撤销所有文件到工作区
git revert `commit` //重做某个commit

7、移除暂存区的修改

git rm --cached `file`

8、本地和远程关联

git branch --set-upstream-to=origin/branch

9、查看分支

git branch
git branch -a
git branch -r

10、合并分支

git merge `branch`
git rebase
//这两种具体的区别可自行查看

11、删除分支

git branch -D `branch` //删除本地分支

12、其他

git tag 
git diff
git stash
....
